# 云爪纪念馆

一个可直接部署到 GitHub Pages 的静态首页原型。

## 发布到 GitHub Pages

1. 在 GitHub 新建一个仓库，例如 `cloud-paw-memorial`。
2. 上传本文件夹中的 `index.html` 和 `style.css`。
3. 打开仓库的 **Settings → Pages**。
4. 在 **Build and deployment** 中选择 **Deploy from a branch**，分支选择 `main`，目录选择 `/ (root)`，然后保存。
5. 等待一两分钟，GitHub 会显示网站地址。

## 发布前要改的内容

`index.html` 最后“开始为 TA 留下星光”的按钮目前使用了示例邮箱 `hello@example.com`。发布前请替换成你的真实联系邮箱，或在后续开发中接入预约表单。
